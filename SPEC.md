## 一、项目定位

**名称**：天线情报追踪系统（Antenna Industry Intelligence Tracker）
**目的**：一站式展示天线行业的市场情报、招标动态、企业图谱、技术趋势和行业新闻
**目标用户**：天线行业从业者、投资者、行业研究员

**访问地址（待部署）**：
- Gitee Pages：`https://maoxj-goal.gitee.io/antenna-tracker/`（待 Gitee 实名认证后开启）
- 本地预览：`http://localhost:3002`

## 三、页面功能规格

### 3.1 首页概览（`/`）

**目的**：快速了解行业整体健康度

**组件**：
- **KPI 指标卡**：活跃招标数、近期中标数、行业新闻数、注册企业数
- **中标趋势折线图**：近6个月中标数量趋势（按月份）
- **市场细分饼图**：6个细分市场的份额占比
  - 基站天线
  - 室分天线
  - 卫星通信天线
  - 5G天线
  - 网通天线
  - 微波天线
- **活跃企业列表**：显示重点企业名称 + 角色标签，点击弹窗显示详情

**数据来源**：`data/market.json` + `data/companies.json`

**状态**：✅ 已实现

---

### 3.2 市场数据（`/market`）

**目的**：展示6大细分市场的规模、份额、增长率

**字段要求**（每条市场数据）：
- 市场规模（单位：亿元）
- 市场占有率（%）
- 年增长率（%）
- 主要驱动因素

**状态**：✅ 已实现

---

### 3.3 企业图谱（`/companies`）

**目的**：展示上中下游供应链企业关系

**供应链分层**：
| 层级 | 说明 | 代表企业 |
|------|------|---------|
| 上游 | 材料与元器件 | 生益科技、华正新材、村田 |
| 中游 | 设计与制造 | 华为、中兴、盛路、亨鑫、通宇、京信、东山精密 |
| 下游 | 运营商与集成商 | 中国移动、中国电信、中国铁塔、中国星网 |

**企业数据字段**（每家企业）：
```json
{
  "id": "string",
  "name": "string",
  "nameEn": "string",
  "role": "string（角色定位）",
  "position": "string（市场定位）",
  "customers": ["string"] | null,
  "highlights": ["string"]（核心竞争力）,
  "location": "string（总部/产地）",
  "isKey": "boolean（是否为重点企业）"
}
```

**交互要求**：
- Tab切换：全产业链总览 / 上游 / 中游 / 下游
- 企业卡片：显示名称、定位、亮点
- 详情弹窗：点击企业卡片弹出详情

**状态**：✅ 已实现（字段完全匹配，无Bug）

---

### 3.4 新闻动态（`/news`）

**目的**：聚合天线行业最新资讯

**内容要求**：
- 每条新闻包含：标题、时间、分类、摘要
- 分类标签：集采 / 5G-A / 卫星通信 / 财报 / 政策 / 其他
- 筛选功能：按分类标签筛选新闻
- 数据条数：≥15条

**状态**：✅ 已实现（筛选功能正常）

---

### 3.5 价格走势（`/prices`）

**目的**：跟踪原材料价格趋势

**原材料品种**：
- 铜（万元/吨）
- 铝（万元/吨）
- PCB（覆铜板指数）
- PTFE（聚四氟乙烯）

**图表要求**：折线图，显示近6个月趋势

**状态**：✅ 已实现

---

### 3.6 标准规范（`/standards`）

**目的**：汇总行业标准

**分类**：
- 国内标准（GB）
- 行业标准（YD/T）
- 国际标准（3GPP / ITU）

**每条标准字段**：
- 标准号
- 标准名称
- 发布机构
- 适用场景

**状态**：✅ 已实现

---

### 3.7 技术页面（`/technology`）

**目的**：展示天线行业主流技术路线和发展阶段

**两大模块**：

#### 模块A：Gartner Hype Cycle 气泡图
- X轴：技术涌现年份
- Y轴：成熟度（1-10）
- 气泡大小：固定（无实际数据含义）
- 气泡颜色：按阶段着色
  - Trigger（触发期）：#8B5CF6
  - Peak（泡沫顶峰）：#EF4444
  - Trough（泡沫低谷）：#F59E0B
  - Slope（稳步爬升）：#3B82F6
  - Plateau（生产成熟）：#10B981
- 点击气泡：跳转到对应技术详情

**技术清单**（7项）：
1. AI天线优化
2. 毫米波天线
3. 透镜天线
4. Massive MIMO
5. LCP基材
6. AiP封装
7. 卫星互联网终端

#### 模块B：技术详情面板
- 主流技术路线
- 代表厂商对比
- 发展方向
- 技术难点
- 预计商用时间线

**状态**：✅ 已实现（Scatter气泡图已修复，点击跳转正常）

## 四、数据维护手册

### 4.1 新闻数据更新
**文件**：`data/news.json`
**频率建议**：每周更新一次
**步骤**：
1. 打开 `data/news.json`
2. 在 `news` 数组头部插入新条目（保持时间倒序）
3. 每条格式：
```json
{
  "id": "news-YYYYMMDD-XX",
  "title": "标题",
  "date": "YYYY-MM-DD",
  "category": "集采|5G-A|卫星通信|财报|政策|其他",
  "summary": "一句话摘要"
}
```

### 4.2 价格数据更新
**文件**：`data/prices.json`
**频率建议**：每月更新
**数据来源**：长江有色金属网、卓创资讯

### 4.3 企业数据维护
**文件**：`data/companies.json`
**字段说明**：严格遵守 3.3 节的字段规范，新增字段需同步更新本文件
